## Publish on GitHub Pages
1. Upload all files/folders to the repository root (keep folder structure).
2. Settings → Pages
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /(root)
3. Open: `https://<username>.github.io/<repo>/`
